Types

JS uses dynamic typing.

There are 6 Primitive Types in JS. They are:

  • UNDEFINED which defines emptiness or lack of existence.. This shouldn't ever be used by the coder.
  • NULL which defines emptiness or lack of existence.. This can be used by the coder.
  • BOOL
  • NUMBER. There is only one number type in JS. This is a floating point number.
  • STRING. A sequence of characters within a pair of ' or ".
  • SYMBOL
